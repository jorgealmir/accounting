# Accounting

API Accounting

### Dependencies
* PHP >= 7.2

### Installation:

If you already have a composer.json file, just add the following dependency to your project:

```json
"require": {
    "mpdf/mpdf": "^8.0"
}
```

### To use the library:
```php
<? php

require __DIR__ . '/vendor/autoload.php';

use Jamartins\Report\Report;

$report = new Report();
```
